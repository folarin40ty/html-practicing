introduction to git and github

Git is widely used in software development, but its applications extend to other fields like writing, design, and research, where version control and collaboration are valuable.

GitHub is a web-based platform that provides a centralized location for version control and collaboration on software development projects. It's built on top of the Git version control system and offers a range of features and tools to facilitate collaborative coding, project management, and community engagement.

Basic Git command

  1. Git init
  2. Git status
  3. Git add .
  4. Git commit -m
  5. Git push
